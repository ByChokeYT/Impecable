Qué es Impeccable

Impeccable es una colección de código abierto de habilidades de agentes (archivos de texto) que se ejecutan localmente en tu herramienta de programación con IA. Las habilidades en sí no recopilan ningún dato, no realizan solicitudes de red y no tienen análisis de ningún tipo.

Análisis del sitio web

El sitio web de Impeccable (impeccable.style) utiliza Google Analytics para comprender los patrones de tráfico (páginas vistas, referencias, país). No se recopila información personal más allá de lo que Google Analytics proporciona por defecto. No se utilizan cookies con fines publicitarios.

Descargas

Cuando descargas un paquete de habilidades desde el sitio web, registramos el evento de descarga (qué paquete y la marca de tiempo) con fines de estadísticas de uso. No se asocia ninguna información personal a estos registros.

Plugin de Claude Code

Cuando se instala como un plugin de Claude Code, Impeccable se ejecuta completamente dentro de tu sesión local de Claude Code. No se envía ningún dato a los servidores de Impeccable. La propia política de privacidad de Anthropic rige la aplicación de Claude Code.

Extensión de Chrome

La extensión Impeccable Chrome DevTools se ejecuta completamente en tu navegador. Toda la detección de antipatrones ocurre localmente en la página que estás inspeccionando. Ningún contenido de página, URL o resultado de detección se envía jamás a ningún servidor externo.

La extensión guarda tus preferencias de reglas (qué detecciones están activadas o desactivadas) utilizando el almacenamiento de sincronización integrado de Chrome (chrome.storage.sync), que sincroniza la configuración entre tus instancias de Chrome a través de tu cuenta de Google. No se almacena ni transmite ningún otro dato.

La extensión solicita los siguientes permisos:

  • activeTab / scripting: para inyectar el script del detector en la página que estás inspeccionando
  • storage: para guardar tus preferencias de reglas
  • webNavigation: para volver a escanear automáticamente cuando navegas a una nueva página
  • Permisos de host (todas las URLs): para que el detector pueda ejecutarse en cualquier sitio web que decidas inspeccionar

GitHub
